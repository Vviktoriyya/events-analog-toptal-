JavaScript Key Code

Key Code Information

event.ctrlKey

`ctrlKey` is a boolean that shows if the "Ctrl" key was pressed (`true`) or not (`false`).

event.key

The value of the key pressed. Accounts for modifiers keys that return CAPS and alternate chars.

event.code

The physical key on the keyboard. Doesn't care if you are holding a modifier like Shift.

event.keyCode

`keydown` is an event triggered when a key is pressed. It provides information about the pressed key, such as `key`, `code`, and `ctrlKey`.
